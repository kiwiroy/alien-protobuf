# NAME

Alien::ProtoBuf - find Google ProtoBuf library

# VERSION

version 0.09

# SYNOPSIS

```perl
use Alien::ProtoBuf;

my $cflags = Alien::ProtoBuf->cflags;
my $libs = Alien::ProtoBuf->libs;

# use $cflags and $libs to compile a program using protocol buffers
```
